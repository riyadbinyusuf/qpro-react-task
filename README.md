This is a basic todo list with filter and basic form builder app. Built with React, React Router, Reqct Query, CSS Module.

Installation:

1. Install dependencies: `npm install`
2. Duplicate env.example, rename it to .env.local, and update the base API URL
3. Start the server: `npm run dev`

Features:

- Todo lists with filters and basic pagination
- Basic dyanmic form builder
- Form previw 

Usage:

- Clone the repository
- Install dependencies
- Set up the environment variables
- Start the server
